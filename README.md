# Password Checker (C)

A simple C program that checks password strength based on defined rules.

## Features
- Minimum length validation
- Character checks
- Simple CLI interface

## How to Run
1. Compile:
   gcc password_checker.c -o password_checker
2. Run:
   ./password_checker

## Concepts Used
- C programming
- Loops & conditions
- Strings
